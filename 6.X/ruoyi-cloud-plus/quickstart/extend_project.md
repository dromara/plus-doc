# 基于 RuoYi-Cloud-Plus 的扩展项目列表
- - -
[扩展项目列表跳转](/ruoyi-vue-plus/quickstart/extend_project.md)

说明：
- 大多数扩展项目的接入思路、功能边界、使用方式，与 Vue 版本是一致的，可直接参考 Vue 版页面
- Cloud 版本接入时，额外关注三类差异：
  - 网关路由是否需要新增对外入口
  - Nacos 配置是否需要新增对应服务配置
  - 新模块是独立微服务，还是并入现有服务模块

如果扩展项目涉及公共登录、文件、短信、第三方授权等通用能力，通常先按 Vue 版文档理解功能，再按 Cloud 的网关与配置中心方式落地即可。
