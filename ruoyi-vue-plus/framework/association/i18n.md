# 国际化方案
- - -

## 前端国际化

参考 RuoYi 前端国际化文档：

- http://doc.ruoyi.vip/ruoyi-vue/document/htsc.html#前端国际化流程

## 后端国际化

- 2.7.0 以上支持后端国际化
- 3.4.0 以上支持 `Validator` 校验国际化

参考 demo 模块 `TestI18nController`。

请求头中增加 `content-language`，值需与国际化资源文件后缀一致，例如 `zh_CN`、`en_US`。

## Validator 校验国际化（Controller）

Controller 参数校验需在类上增加 `@Validated`。

![输入图片说明](https://foruda.gitee.com/images/1678976741834729507/6c19b9cc_1766278.png "屏幕截图")<br>
